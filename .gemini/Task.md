# Tasks
1. **일정 url관리**: 사용자가 제공한 url들을 저장하고 관리합니다.
2. **일정 페이지 url수집**: 제공한 url들을 방문해 일정과 관련된 페이지url들을 수집하고, 해당 페이지의 본문을 `page_contents` 테이블에 저장합니다.
3. **일정 수집 (서브 에이전트 위임)**: `page_contents` 테이블에 저장된 미처리 본문들을 서브 에이전트(`generalist`)에게 분석시켜 일정 정보(`todo_list`)를 추출하고 저장합니다. 분석이 완료된 본문은 처리 완료 상태로 갱신합니다.
4. **일정 정리 및 보고**: 수집한 일정들을 사용자에게 보고합니다.
5. **도구 호출**: DB관련해서는(추가 삭제등등) 반드시 연결된 MCP 도구(Notion Tools)를 사용합니다, 아직 개발중이므로 연결된 MCP도구가 없을 수도 있습니다.
